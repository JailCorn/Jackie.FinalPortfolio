E-Commerce Website Prototype

This project was done for my CM316 User Experience course in my Junior year at UH West Oahu in Spring 2024. This was my first time using the web design app Figma and the graphic design app Canva. I'm decently satisfied with how this project came out. In the future, I would look at more inspiration to create a design that balances elegance and cuteness in a more unique way. I had to downsize the design a lot because it got confusing as I started categorizing each page and seeing how they flowed. We didn't create site maps for this assignment, but I think we should have.

This course taught me the basic sections of an e-commerce website homepage that users often overlook. My instructor also taught me to be as specific as possible when designing.

E-Commerce Website Process

For this project, we had to create our own e-commerce brand, which is convenient because this is something that I have an interest in anyway. The JCA cat logo is something I came up with on the fly, but I ended up using it as my logo for my personal business because I liked the way it turned out!

We worked on the designs of our websites using Canva. I still prefer using Illustrator for graphic design, but Canva was easy enough to use. The website copy was something I struggled with because it felt like I was writing for everyone else except myself. Trying to keep things sounding professional just didn't feel true to myself, so I would definitely stick to writing copy that I actually enjoy.

For inspiration, I gravitated towards some artists I look up to, such as @mg3kiryu on Twitter (X), who runs their e-commerce site, umvelt.fish. I also looked to @ohmonah on Twitter, who runs ohmonah.shop. For help with other design features, I also looked at popular e-commerce sites like Amazon and Etsy.

It was really nice seeing a site made up entirely of my own art!
